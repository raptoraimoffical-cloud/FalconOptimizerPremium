# QA Tests

## Downloads
1. Launch the app.
2. In the sidebar, click "Project files (npm start)".
3. Verify the browser opens the GitHub project zip URL.
4. Click "Installer" and verify the releases page opens.

## Home Preview
1. Open `index.html` in a browser preview.
2. Confirm the Home panel shows CPU/GPU/RAM/OS values (preview mode labels are acceptable).
