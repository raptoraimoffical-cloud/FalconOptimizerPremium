Falcon Optimizer - NV Clean Install Helper
=========================================

This folder is used by the "NV Clean Install (helper)" action under:
  Hardware -> GPU

To use it:
  1) Download NVCleanstall.exe (or your preferred nvclean helper) manually.
  2) Place the EXE file in this folder.
  3) In Falcon Optimizer, run the "NV Clean Install (helper)" action.
     The optimizer will open this folder so you can launch the tool.

The optimizer does NOT bundle or download NVIDIA tools automatically.
